# Задача.

Задание можно выполнить в кофигурации 1с розница, 1с ут или унф на выбор
1) В табличной части документа Расходная накладная добавить колонку с выводом штрихкодов по номенклатуре. Штрихкод должен заполняться при подборе номенклатуры в табличной части.
2) Для этого документа добавить новую печатную форму УПД с ШК
Общие требования:
• Макет новой печатной формы должен быть такой же как типовой макет УПД. Добавляется только в табличную часть колонка со штрихкодами вместо Артикула.
• Добавить новую печатную форму в список печатных форм документа «Расходная накладная» , чтобы была возможность формирования через типовую кнопку «Печать с предварительным просмотром».
Примечание:
- Штрихкод — берется из колонки «Штрихкоды» табличной части документа «Расходная накладная»
- Остальная часть макета новой печатной формы такая же как у типовой печатной формы УПД.

Доработку нужно сделать через расширение.
